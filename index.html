В данный момент обучаюсь на Hexlet по профессии Frontend-разработчик. Во время обучения на данный момент выполнено 2 проекта:

Brain-games - представляет собой набор из пяти консольных игр, разработанных по принципу популярных игр для прокачки мозга. Стек: Чистый JavaScript

Difference calculator - это консольное приложение, которое определяет разницу между двумя структурами данных "json" и "yml/yaml". Вывод поддерижвает 3 формата, так же можно использовать как библиотеку. Стек: JavaScript, библиотека Lodash, для написания тестов библиотека Jest, Commander.js - модуль Nodejs, который предоставляет мощные функции для ввода данных командной строки и анализа параметров.

RSS aggregator - платформа для чтения RSS ленты. Валидация формы с помощью yup, Ajax запросы выполнены с помощью Axios, сервис парсит и рендерит данные, так же обрабатывает ошибки и обновляет фиды ассинхронно. Проект задеплоен на vercel.app. Проект реализован следуя концепции MVC. Стек: Webpack, Bootstrap 5, Validator, i18next, Axios.
